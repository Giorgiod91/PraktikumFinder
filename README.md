**Beschreibung:**  
PraktikumFinder ist eine kleine Web-App, die angehenden Umschülern zum Fachinformatikern Anwendungsentwicklung , **passende Betriebe für das Pflicht Praktikum** zu finden.  
Ziel des Projekts ist es, eine Plattform bereitzustellen, die es derzeit **so noch nicht gibt**, sodass zukünftige Umschüler direkt sehen können, **wo sie sich bewerben können**.

## Funktionen
- Anzeige einer Liste von Firmen in Hannover und Umgebung, die Fachinformatiker ausbilden
- Filter nach Ort, Branche oder Ausbildungsangebot
- Direkte Links zu Bewerbungsseiten / Kontaktdaten
- Favoritenfunktion, um interessante Firmen zu speichern
- Map-Ansicht für den Standort der Firmen

## Zielgruppe
- Umschüler, die eine Ausbildung zum Fachinformatiker Anwendungsentwicklung beginnen möchten
- Studierende oder Praktikanten, die nach passenden Ausbildungsbetrieben suchen

## Technologie
- **Backend:** Python
- **Frontend:** HTML / CSS / JavaScript
- **Deploy:** Render (Backend)
